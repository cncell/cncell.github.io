Privacy Policy (Политика конфиденциальности)

1. Сбор данных

  • 1.1. Бот может собирать минимальные данные о пользователях для обеспечения своей функциональности.
  • 1.2. Собираемые данные могут включать:
    • Имя пользователя и его уникальный идентификатор (Discord ID).
    • Содержимое сообщений, используемых для взаимодействия с командами бота.
    • Данные о сервере, такие как идентификаторы каналов и ролей.

2. Использование данных

  • 2.1. Все собираемые данные используются исключительно для выполнения команд и предоставления функционала бота.
  • 2.2. Бот не использует данные для анализа или передачи третьим лицам.

3. Хранение данных

  • 3.1. Данные сохраняются временно и удаляются, как только они становятся ненужными для выполнения задач.
  • 3.2. Постоянное хранение данных может осуществляться только в случаях, предусмотренных функционалом бота (например, уровни, напоминания и т. д.).

4. Передача данных

  • 4.1. Данные не передаются третьим лицам, кроме случаев, когда это требуется законом.
  • 4.2. Если бот интегрируется с внешними API (например, для получения изображений или данных), передача данных осуществляется минимально необходимым объемом.

5. Безопасность данных

  • 5.1. Все данные защищены и используются только внутри инфраструктуры бота.
  • 5.2. Разработчик принимает меры для предотвращения утечки данных, но не несет ответственности за внешние атаки или утраты, вызванные использованием сторонних платформ.

6. Права пользователей

  • 6.1. Пользователь имеет право запросить удаление своих данных, отправив сообщение разработчику.
  • 6.2. Пользователь имеет право узнать, какие данные о нем хранятся ботом.

7. Изменения в политике

  • 7.1. Политика конфиденциальности может быть изменена без предварительного уведомления.
  • 7.2. Обновленная версия политики вступает в силу с момента ее публикации.

8. Контакты

  • Если у вас есть вопросы или жалобы по поводу использования ваших данных, свяжитесь с разработчиком: Discord: hhere.
